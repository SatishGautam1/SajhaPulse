# SajhaPulse – The Smart Information Hub 🇳🇵

**SajhaPulse** is a modern, high-performance Android utility application designed for the Nepali market. Developed under the **Night Tech** brand, it serves as a "Zero-Latency" gateway to essential daily information, from real-time market rates to viral local content.

## 🚀 Key Features
*   **Live Tickers:** Real-time Gold, Forex, and Remittance rates via Firebase Realtime Database.
*   **Daily Bazar:** Smart price tracking for vegetables and essential commodities.
*   **Chautari Feed:** A curated space for memes and trending viral videos.
*   **Micro-Remittance Calculator:** Compare NPR conversion rates across major providers (IME, City Express, etc.) instantly.
*   **Digital Ad Portal:** A dedicated B2B space for local businesses to reach users.

## 🛠 The Modern Power Stack
*   **Language:** Java (Native Android Studio)
*   **UI/UX:** Material Design 3 (M3) with `androidx.core:core-splashscreen`.
*   **Backend:** Firebase (Auth, Firestore, Realtime Database, Cloud Storage).
*   **Architecture:** Hybrid Spark Model (Firestore for structured data + Room for offline caching).
*   **Networking:** Retrofit 2 & Jsoup for web scraping local data.

## 📱 User Experience
*   **One-Tap Login:** Modern Google Sign-In via Credential Manager.
*   **Dual-Nav System:** Sidebar for personalized settings and Bottom Navigation for core modules.
*   **Offline-First:** All critical data is cached locally via Room Database to ensure the "Night Tech" experience works even without an internet connection.
